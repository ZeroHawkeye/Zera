## 代码块

支持语法高亮的代码块：

```typescript title="example.ts"
interface User {
  id: string;
  name: string;
  email: string;
}

async function getUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
}
```

## 步骤

<Steps>
  <Step>
    ### 安装依赖
    运行 `npm install` 或 `pnpm install` 安装项目依赖
  </Step>
  
  <Step>
    ### 配置环境
    复制 `.env.example` 到 `.env` 并填写配置
  </Step>
  
  <Step>
    ### 启动开发
    运行 `task dev` 启动开发服务器
  </Step>
</Steps>
