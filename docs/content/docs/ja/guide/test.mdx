## コードブロック

構文ハイライト付きのコードブロック：

```typescript title="example.ts"
interface User {
  id: string;
  name: string;
  email: string;
}

async function getUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
}
```

## ステップ

<Steps>
  <Step>
    ### 依存関係のインストール
    `npm install` または `pnpm install` を実行してプロジェクトの依存関係をインストール
  </Step>
  
  <Step>
    ### 環境設定
    `.env.example` を `.env` にコピーし、設定を記入
  </Step>
  
  <Step>
    ### 開発の開始
    `task dev` を実行して開発サーバーを起動
  </Step>
</Steps>

## テーブル

| コマンド | 説明 |
|------|------|
| `task dev` | 開発環境を起動 |
| `task proto` | protoワークフローを実行 |
